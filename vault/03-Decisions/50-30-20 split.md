# 50/30/20 split for small datasets

**Status:** accepted (revisit once dataset > 300 images)
**Area:** data

## Context

Default Roboflow split gave only 3 validation images. First run reported mAP@50 ~28.6% — looked catastrophic — but per-image detection was actually fine.

## Decision

Use 50/30/20 train/valid/test until the dataset is large enough to absorb a 70/20/10 split reliably.

## Why

With 3 val images, a single false negative tanks mAP by 33%. Misleadingly low numbers are a bad signal for decision-making.

## Consequences

- Smaller training set, but trustworthy metrics.
- Revisit at ~300+ annotated images.
